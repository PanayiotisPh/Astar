# Astar
## What it solves
```bash=
Given a puzzle BBBWWWE the Astar should sort the puzzle by moving all the B to the right site of the puzzle
and all the W to the left site, independently of the position of E.
```
## How to use
```bash=
User gives the puzzle thought the command line and the program gives the route
to the solution including description of the changes on each state, the
heuristic and the state of puzzle. At the end it prints the total of created
nodes and time and memory needed.
```
